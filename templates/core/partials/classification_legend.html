Classification legend
Referenced
Matched to a VM/CT disk reference in the latest Proxmox inventory.
Likely orphan
No matching Proxmox reference was found, and all expected consumers were scanned.
Blocked
pve-helper could not safely decide because one or more expected consumers were missing.
Unknown
Regular file, but not enough evidence to call it referenced or orphaned.
Infrastructure
Directory or datastore structure that is not treated as an orphan candidate.
Proxmox content
Recognized Proxmox-managed content that should be handled conservatively.
Import source
OVA/OVF metadata, manifests, or loose disks recognized as VM import sources.
Trash
File is already inside pve-helper's Recycle Bin area.
